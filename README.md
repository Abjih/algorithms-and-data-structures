# algorithms-and-data-structures
================================================================================ ЛАБОРАТОРНЫЕ РАБОТЫ: СТРУКТУРЫ ДАННЫХ И АЛГОРИТМЫ
Репозиторий содержит реализацию классических структур данных и алгоритмов
на C++ с полным покрытием unit-тестами (Google Test). Каждая лабораторная
работа — самостоятельный модуль с единой структурой проекта.
СТРУКТУРА РЕПОЗИТОРИЯ
mp2-lab1-bitfield/
mp2-lab2-matrix/
mp2-lab3-stack_rpn/
mp2-lab4-polynomial/
mp2-lab5-tables/
Каждая папка содержит:
include/    — заголовочные файлы с реализацией
gtest/      — фреймворк Google Test
test/       — unit-тесты
samples/    — примеры использования
sln/        — решение Visual Studio
================================================================================ 
ЛАБОРАТОРНАЯ РАБОТА 1: БИТОВОЕ ПОЛЕ
Реализация класса TBitField для хранения и манипуляции битовыми массивами
произвольной длины.
Основные операции:
Установка, сброс и чтение отдельных битов
Битовые операции: И, ИЛИ, НЕ, сравнение
Ввод/вывод через потоки
Ключевые особенности:
Хранение битов в массиве unsigned int с побитовой адресацией
Экономия памяти: один элемент массива хранит 32 бита
================================================================================ 
ЛАБОРАТОРНАЯ РАБОТА 2: МАТРИЦЫ И СПЕЦИАЛЬНЫЕ ФОРМАТЫ ХРАНЕНИЯ
ДИНАМИЧЕСКАЯ МАТРИЦА
Шаблонный класс TDynamicMatrix<T><T>
Операций с векторами и скалярами (+, -, *, скалярное произведение)
Матричных операций: сложение, вычитание, умножение матриц,
умножение матрица-вектор
Move-семантика и RAII
СПЕЦИАЛЬНЫЕ МАТРИЦЫ
Разреженные матрицы — матрицы, в которых большинство элементов равны нулю.
Для их хранения реализованы три формата, позволяющие экономить память
и ускорять операции за счет хранения только ненулевых элементов:
Формат      Класс
COO         CRD<T>
CSR         CSR<T>
Ленточный   BandMatrix<T>
Все форматы поддерживают:
Конвертацию из плотной матрицы и обратно
Умножение матриц (через промежуточное преобразование в плотный формат)
Доступ к элементам по индексам

ЛАБОРАТОРНАЯ РАБОТА 4: ПОЛИНОМЫ
Реализация класса Polynomial для работы с многочленами от трех переменных
(x, y, z) с однозначными степенями (0–9).
Структура данных: односвязный список мономов, упорядоченных по убыванию
степени.
Операции:
Сложение, вычитание, умножение полиномов
Автоматическое приведение подобных членов и удаление нулевых коэффициентов
Парсинг полинома из строки (поддержка записи вида 3x^2y^1z^0, (xyz)^3, xyz)
================================================================================ 
ЛАБОРАТОРНАЯ РАБОТА 5: АССОЦИАТИВНЫЕ КОНТЕЙНЕРЫ
Реализация 6 структур данных для хранения пар «ключ-значение»
(ключ — строка, значение — полином):
Структура                        Класс                Поиск     Особенности
Неупорядоченная таблица          UnorderedArrayTable  O(n)      Линейный поиск, на массиве удаление через swap-with-last
Упорядоченная таблица            OrderedArrayTable    O(log n)  Бинарный поиск, на массиве вставка со сдвигом
Упорядоченная таблица            OrderedListTable     O(n)      Dummy-узел, на списке упорядоченность по ключу
Хеш-таблица                      HashTableChains      O(1) ср.  Разрешение коллизий (метод цепочек) списками
Хеш-таблица                      HashTableDouble      O(1) ср.  Открытая адресация, (двойное хеширование) рехеширование при load>0.5
Красно-черное дерево             RBTree               O(log n)  Самобалансирующееся BST, вставка с перебалансировкой
Скип-лист                        SkipList             O(log n)  Вероятностная структура, уровни через coin flip
